HTML

1. basic_info

  • <태그명>: 열리는 태크
  • : 닫히는 태크
  • 부모 태그 - 자식 태그
  • 웹페이지
  • 웹사이트: 웹페이지들의 그룹
  • 웹브라우저: 웹페이지 검색 프로그램
  • 웹서버를 직접 설치하거나 웹호스팅 업체를 이용하여 내가 만든 웹페이지를 인터넷에서 검색 가능
  • 웹호스팅 업체: 웹서버를 제공해주는 회사

2. tag

  • br: line break의 br을 따옴 (개행)
  • strong: 굵은 글씨
  • u: 밑줄 긋기
  • h1~h6: 제목 ( 숫자가 낮을수록 중요성 업 == 글자크기 증가 )
  • img: 이미지 삽입 태그
  • -> src속성을 이용하여 사진을 지정 (방법은 아래와 같음)
  • -> src="웹브라우저에서 사진의 주소" or src="현재 소스코드가 있는 폴더에 있는 사진의 이름"
  • li: list의 준말 -> 목차를 나타냄
  • ol: orderded list의 줄임말 li 자식태그의 내용의 번호를 부여 O
  • ul: unorderded list의 줄임말 li 자식태그의 내용의 번호를 부여 X
  • head: 본문 설명
  • body: 본문 내용
  • html: head와 body태그를 모두 감쌈
  • !doctype html: 웹 페이지가 html로 만들어졌다는 것을 웹브라우저에 알림
  • a:anchor(닻)의 준말
  • ->href(hypertext reference) 속성과 함께 사용하여 원하는 내용에 hyperlink 걸어둠
  • ->title 속성으로 tooltip (커서를 올려놓았을 때 뜨는 창)에 원하는 내용 표시
  • ->target="_blank" 속성 사용시 새로운 창에 웹페이지 염
